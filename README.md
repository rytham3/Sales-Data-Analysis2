# Sales Data Analysis Project

## Overview
Analyzed real-world sales data to understand business performance across regions, products, and customer types.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib

## Key Analysis
- Monthly and yearly sales trends
- B2B vs B2C performance comparison
- Product-wise and location-wise analysis
- Profitability analysis using gross margin

## Outcome
Extracted actionable insights to support business and sales strategy decisions.
